동작 파라미터화를 이용해서 변화하는 요구사항에 효과적으로 대응하는 코드를 구현 할 수 있음.
익명 클래스로 다양한 동작을 구현할 수 있지만 코드가 깔끔하지 않다.
람다 표현식을 이용하여 이름이 없는 함수이면서 메서드를 인수로 전달 할 수 있다.
또한 람다 표현식과 함께 위력을 발휘하는 새로운 기능인 메서드 참조를 확인 한다.

1. 람다란 무엇인가?
람다 포현식은 메서드로 전달 할 수 있는 익명 함수를 단순화 한 것.

CRunnable 코드 확인.

함수디스크립터
함수형 인터페이스의 추상 메서드 시그니처는 람다 표현식의 시그니처를 가리킨다.
함수형 인터페이스를 인수로 받는 메서드에만 람다 표현식 사용 가능

람다 활용 : 실행어라운드 패턴
